Technologies Used

PostgreSQL

Implemented Postgresql for creating an ERP system including faculty, students and attendance data.Postgresql is highlt reliable and scalable.

Keras facenet

Keras Facenet is a deep learning-based facial recognition model. Used it to classify faces and to get 128-dimensional vectors of faces.

Numpy

Numpy is a library for efficient numerical computation in Python. We used Numpy to get 128-dimensional vectors of faces, which enabled us to represent facial features in a compact and meaningful way.

Flask

Flask is a lightweight and flexible web framework that enables rapid development of web applications. Thus, used Jinja templating engine to create dynamic web pages.
